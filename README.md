##  Overview

**RapidAid** is a web-based healthcare AI voice assistant that allows users to speak or type health-related questions and receive intelligent, conversational responses powered by Google Gemini. It aims to provide fast, accessible, and reliable first-response health guidance — directly from your browser.

>  **Disclaimer:** RapidAid is not a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider for medical concerns.

---

##  Features

- 🎙️ **Voice Input** — Speak your health questions naturally using your device microphone
- 🤖 **Gemini AI Responses** — Intelligent answers powered by Google's Gemini language model
- 💬 **Conversational Interface** — Chat-style UI for a smooth, intuitive experience
- ⚡ **Fast & Lightweight** — Built with Vite for near-instant load times
- 🌐 **Browser-Based** — No installation required; runs entirely in the browser
